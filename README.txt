StockExchange
------------------------------------------
1.  Requirements

     Python version : 2.7

     Exemple of IDE you can use : PyCharm

2.   Get finance API by run :  pip install tushare

3.   Solve package requirements :     pip install -r requirements.txt

4.   run with :   python manage.py runserver    

5.   main page :   127.0.0.1:8000

6.   documentation page:    127.0.0.1:8000/admin/doc

7.   administration page:      127.0.0.1:8000/admin

Note :   you can query any stock beyond the listed as long as you know the stock code
